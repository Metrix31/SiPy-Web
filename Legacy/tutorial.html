SiPy Tutorial (Deutsch)

Was ist SiPy?

SiPy ist eine einfache, moderne Programmiersprache, die du direkt im Browser ausführen kannst. Sie basiert auf JavaScript, besitzt aber auch eine vollständige Python‑Version sowie einen lokalen SiPy‑Editor für Windows, die beide auf der GitHub‑Seite verfügbar sind.

SiPy unterstützt Variablen, Eingaben, Schleifen, Bedingungen und seit kurzem auch Module über import("…").

Variablen

a = integer(10)
b = string("Hallo")
writeln(a)
writeln(b)

Eingabe

name = string(getln("Wie heißt du?"))
writeln(name)

Schleifen

loop(3, "writeln('Hallo Welt')")

Bedingungen

ifcase(a.eq(integer(10)), "writeln('a ist 10')", "writeln('a ist nicht 10')")

Neue Funktionen in SiPy

📦 Module importieren

SiPy unterstützt jetzt Module, die mit import("name") geladen werden.

import("math")
import("strings")

🧮 Math‑Modul

import("math")
writeln(math.pi())
writeln(math.sqrt(16))
writeln(math.add(10, 20))

🔤 Strings‑Modul

import("strings")
writeln(strings.upper("hallo"))
writeln(strings.lower("HALLO"))

SiPy Tutorial (English)

What is SiPy?

SiPy is a simple, modern programming language that runs directly in your browser. It is based on JavaScript, but also has a full Python version and a local SiPy Editor for Windows, both available on the GitHub page.

SiPy supports variables, input, loops, conditions, and now also modules via import("...").

Variables

a = integer(10)
b = string("Hello")
writeln(a)
writeln(b)

Input

name = string(getln("What is your name?"))
writeln(name)

Loops

loop(3, "writeln('Hello World')")

Conditions

ifcase(a.eq(integer(10)), "writeln('a is 10')", "writeln('a is not 10')")

New Features in SiPy

📦 Importing Modules

SiPy now supports modules that can be loaded using import("name").

import("math")
import("strings")

🧮 Math Module

import("math")
writeln(math.pi())
writeln(math.sqrt(16))
writeln(math.add(10, 20))

🔤 Strings Module

import("strings")
writeln(strings.upper("hello"))
writeln(strings.lower("HELLO"))
