Was ist SiPy?

SiPy ist eine einfache Programmiersprache für kleine Programme direkt im Browser. Du schreibst Code links im Editor, führst ihn mit Run oder Strg+Enter aus und siehst das Ergebnis rechts im Output.

Variablen und Typen

Variablen werden mit = gespeichert. Werte können als Zahl, Text oder Wahrheitswert angelegt werden.

a = integer(10)
b = floater(2.5)
text = string("Hallo")
aktiv = boolean(true)
writeln(a)
writeln(text)

Ausgabe und Eingabe

writeln() schreibt eine neue Zeile in den Output. getln() fragt eine Eingabe ab.

name = string(getln("Wie heißt du?"))
writeln("Hallo " + name.value)

Rechnen und Vergleichen

SiPy-Werte haben Methoden zum Rechnen und Vergleichen.

a = integer(10)
b = integer(3)
writeln(a.add(b))
writeln(a.sub(b))
writeln(a.mul(b))
writeln(a.div(b))
writeln(a.gt(b))

Schleifen und Bedingungen

loop() wiederholt Code. ifcase() führt je nach Bedingung einen von zwei Befehlen aus.

zahl = integer(7)
loop(3, "writeln('Hallo')")
ifcase(zahl.gt(5), "writeln('größer als 5')", "writeln('kleiner oder gleich 5')")

Module

Die Module math und strings sind normal eingebaut und sofort verfügbar. import("math") und import("strings") funktionieren weiterhin, sind aber nicht nötig. 

writeln(math.pi)
writeln(math.sqrt(16))
writeln(math.add(10, 20))
writeln(strings.upper("hallo"))
writeln(strings.lower("HALLO"))

Komplettes Beispiel

Dieses Beispiel nutzt Variablen, ein eingebautes Modul, eine Bedingung und Output.

radius = integer(4)
fläche = math.pi.mul(radius.mul(radius))
writeln("Fläche:")
writeln(fläche)
ifcase(fläche.gt(40), "writeln('große Fläche')", "writeln('kleine Fläche')")
