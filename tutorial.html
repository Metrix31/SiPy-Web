SiPy Tutorial (Deutsch)

Was ist SiPy?

SiPy ist eine einfache, moderne Programmiersprache, die du direkt im Browser ausführen kannst. Sie basiert auf JavaScript, hat aber eine Python‑ähnliche Syntax.

Variablen

a = integer(10)
b = string("Hallo")
writeln(a)
writeln(b)

Eingabe

name = string(getln("Wie heißt du?"))
writeln(name)

Schleifen

loop(3, "writeln('Hallo Welt')")

Bedingungen

ifcase(a.eq(integer(10)), "writeln('a ist 10')", "writeln('a ist nicht 10')")

SiPy Tutorial (English)

What is SiPy?

SiPy is a simple, modern programming language you can run directly in your browser. It is based on JavaScript but has a Python‑like syntax.

Variables

a = integer(10)
b = string("Hello")
writeln(a)
writeln(b)

Input

name = string(getln("What is your name?"))
writeln(name)

Loops

loop(3, "writeln('Hello World')")

Conditions

ifcase(a.eq(integer(10)), "writeln('a is 10')", "writeln('a is not 10')")
